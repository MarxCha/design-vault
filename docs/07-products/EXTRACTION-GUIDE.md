# Productos Open Source — Guía de Extracción

## Patrones de UI/UX Replicables

### De cal.com
- Booking flow UX: selección de hora → confirmación → éxito
- Settings layout con sidebar + content area
- Integration cards grid
- Responsive calendar component

### De Dub.co
- Link analytics dashboard con sparklines
- Onboarding flow multi-step
- Settings con toggle sections
- Toast notifications elegantes

### De Payload CMS
- Admin UI con sidebar de colecciones
- Field editor con live preview
- Rich text editor integrado
- Responsive data tables con bulk actions

### De Penpot
- Canvas-based design editor
- Layer panel con drag-and-drop
- Properties panel con inputs numéricos
- Toolbar con tool groups y shortcuts

## Principio Clave
Estos productos demuestran que el diseño excepcional no es solo 
para landing pages. La UI de producto (formularios, tablas, 
settings, dashboards internos) TAMBIÉN merece atención al detalle.
