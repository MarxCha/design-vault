# Dashboards — Guía de Extracción

## Componentes Clave

### De Tremor (Copy-Paste Dashboard)
- KPI Cards con sparklines
- Area/Bar/Line/Donut charts con defaults elegantes
- Progress bars y badges
- Data tables con sorting

### De shadcn-admin
- Collapsible sidebar con submenus
- Command palette (Cmd+K) 
- Chat interface integrado
- Data tables server-side con filtros

### De Tabler
- Sidebar transparente con backdrop blur
- Card layouts con atención tipográfica
- Form layouts completos
- Dashboard homepage layouts (2 variantes)

### De Flowbite Admin
- CRUD layouts completos
- Two-column dashboard layouts
- Widget cards con ApexCharts
- Calendar y Kanban views
